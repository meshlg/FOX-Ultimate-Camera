# FOX Ultimate Camera

**Version:** 5.05.2146

**Author:** meshlg (@LiminalFox)

**ESOUI:** https://www.esoui.com/downloads/info4009-FOXUltimateCamera.html

## Description

FOX Ultimate Camera is an immersive camera addon for The Elder Scrolls Online that refines zoom, Field of View (FOV), and camera offset behaviors. It introduces smooth transitions, context-aware adjustments (combat, mounted, stealth), and dialog camera tweaks to enhance your gameplay experience.

## Features

- **Smooth Transitions:** seamless smoothing for camera zoom, FOV changes, and offset adjustments.
- **Context Sensitivity:** Automatically adjusts camera sensitivity and zoom speed based on your state (Combat, Mounted, Sprinting, Stealth).
- **First Person Mode:** 
  - Auto-switch to 1st person in interiors, combat, or when mounted.
  - "True First Person" toggle with configurable delays.
- **Camera Offsets:**
  - Define custom X/Y camera offsets (over-the-shoulder view).
  - Save and load up to 3 offset presets.
  - On-screen visual indicator for offset adjustments.
- **Field of View (FOV):**
  - Set distinct FOV values for 1st person and 3rd person views.
  - Contextual FOV: Different FOV settings for Combat, Mount, and Interiors.
- **Weapon Sheathing:** Auto-sheath weapons when out of combat (with configurable delay).
- **Interaction Guard:** Prevents camera changes during interactions (e.g., harvesting, talking to NPCs).

## Installation

1. Download the latest release.
2. Extract the `FOXUltimateCamera` folder into your ESO AddOns directory:
   - **Windows:** `Documents\Elder Scrolls Online\live\AddOns\`
3. Launch the game and enable the addon in the **Addons** menu.

## Dependencies

- **LibAddonMenu-2.0** (Optional, required for Settings menu)

## Chat Commands

- `/foxuc` - Open the addon settings panel.

## Keybinds

Bind these in **Controls > Addon Keybinds > FOX Ultimate Camera**:
- **Camera Move:** Left, Right, Up, Down
- **Reset:** Reset camera to center
- **Swap Side:** Toggle shoulder view (Left/Right)
- **Presets:** Save/Load Presets 1-3
