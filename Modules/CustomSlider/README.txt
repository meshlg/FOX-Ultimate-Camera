FOXUltimateCamera - CustomSlider

This module registers a custom LibAddonMenu-2.0 slider widget "fox_slider".

Features:
- Classic horizontal slider bar.
- MIN / CURRENT / MAX labels rendered below the bar.
- Optional decimals, clamping and custom value formatting via sliderData.valueFormat.

Usage in Settings.lua:
- Resolve control type using addon:GetCustomSliderControlType() helper.
- Use it instead of "slider" for specific options.
